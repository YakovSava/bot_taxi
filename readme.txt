В данный момент бот полностью функционирует (даже смайлики есть!) и может приносить доход как в Vk Pay, так и на Qiwi.

========= Токены =========
Необходимо получить всего 3 токена:
- От группы Вк
- Dadata
- Qiwi p2p (копировать только секретный ключ)

========= Редактирование параметров =========
Есть файл под названием "parameters.json" который необходимо отредактировать под ваши нужды. Что там нужно менять?
{"admin": 0, "count": 0, "group_id": 0}
admin - ID`админа группы. Нужен что бы уведомлять об оплатах
count - Редакиируемо в ВК командой: admin change (Число)
group_id - ID группы (для функционала ВК)

========= Admin команды =========
admin change (число) - меняет количество денег необходимое для взятия заказа
admin histogram - Получить гистограммы (может не работать)
admin get - в Cache сохраняет таблицу (можно открыть в Excel, но в Excel покажет какую-то белиберду т.к. кодирует по другому)
